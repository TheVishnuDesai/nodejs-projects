# Node.js Mini Projects

A collection of simple Node.js projects for practising file system, system monitoring, and backend basics.

---

## 01. File Manager

A command-line tool for basic file system operations.

**Features:**
- Create folders and files  
- Write to files  
- Delete files and folders  
- List directory items  
- Exit the application  

---

## 02. CPU & Memory Monitor

A real-time system monitor for the terminal using Node.js.

**Features:**
- Live CPU usage per core  
- Live memory usage  
- Updates every second  
- Clean console output  

---

## 03. Expenses Tracker API

A simple HTTP server for managing expenses using a JSON file.

**Features:**
- Home and About routes  
- Add expenses (POST)  
- View expenses (GET)  
- File-based data storage  
- Runs on port **3000**

---

### Tech Stack
- Node.js
- Built-in modules: `fs`, `http`, `os`
